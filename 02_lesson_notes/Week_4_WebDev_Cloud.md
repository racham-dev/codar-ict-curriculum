# Week 4: Website Development & Cloud Computing

## Session 7: Website Development Basics
- The 3 Pillars of the Web:
  - HTML: Structure (Bricks).
  - CSS: Style (Paint).
  - JavaScript: Behavior (Electricity).
- CMS (Content Management System): WordPress, Joomla.
- Inspect Element: How to view any website's code.

## Session 8: Introduction to Cloud Computing
- What is Cloud? Storing/accessing data via remote servers.
- 3 Service Models:
  - SaaS (Software as a Service): Google Docs, Zoom.
  - PaaS (Platform as a Service): Google App Engine.
  - IaaS (Infrastructure as a Service): AWS, Azure.
- Benefits: Scalability, Cost efficiency, Global access.
