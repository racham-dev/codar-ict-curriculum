# Week 1: Introduction to ICT & Hardware

## Session 1: Introduction to ICT
- Definition: ICT is the ecosystem of hardware, software, networks, and cloud.
- Evolution: 5 Generations (Vacuum Tubes → Transistors → ICs → Microprocessors → AI).
- Why it matters: Banking, Remote Work, Telemedicine, E-learning.

## Session 2: Computer Hardware Basics
- CPU: The "Brain" (does all math and logic).
- RAM: "Short-Term Memory" (clears when shut down).
- SSD/HDD: "Long-Term Memory" (stores files permanently).
- Peripherals: Input → Processing → Output → Storage.
