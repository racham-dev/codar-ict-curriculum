# Week 2: Computer Software & Networking

## Session 3: Computer Software Basics
- System Software (The "Landlord"): Windows, macOS, Linux, Android, iOS.
- Application Software (The "Tenants"): Chrome, Word, WhatsApp, Zoom.
- 2026 Trend: Cloud-based apps (Google Docs, Canva).

## Session 4: Introduction to Networking
- LAN (Local Area Network): Small area (office/home).
- WAN (Wide Area Network): Large area (The Internet).
- WLAN (Wireless LAN): Wi-Fi networks.
- IP Address: Your device's "Home Address".
- DNS: The "Phonebook" of the internet.
